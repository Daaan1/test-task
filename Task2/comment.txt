N - ошибки
M - ворнинги

Представим N и M в виде плоскости, где ось X - M, а ось Y - N. Таким образом, для исправления всех багов необходимо попасть из точки (M, N) в точку (0, 0).
Исходя из условий задачи, существует 4 возможных действия, которые можно сделать за один коммит:
1) Если Петя исправляет одну фатальную ошибку за коммит, то вместо неё появляется одна новая (N-1; N+1).
2) Если Петя исправляет ворнинг в коммите, то вместо него появляется два новых ворнинга (M-1; M+2).
3) Если Петя исправляет два ворнинга за один коммит, то появляется одна новая фатальная ошибка (M-2; N+1). 
4) Если Петя исправляет сразу две фатальные ошибки за один коммит, то не появляется ничего нового (N-2).

Для исправления всех багов за меньшее число коммитов необходимо в первую очередь уменьшать значения N и M с помощью действий 3 и 4.
Далее, если M равняется одному, то можно использовать сначала действие 2, а затем 3 для уменьшения значения M на один.
Когда значение M будет равняться нулю, а значение N - одному, можно использовать дейстие 2 два раза и действие 3, чтобы значение N стало равно двум, а затем действием 4 уменьшить N до нуля.
Таким образом, N и M будут равняться нулю.
